UTPP
UTPP - A New Generation of UnitTest++

This is a test framework based on UnitTest++.

Latest version can be downloaded from GitHub.

There is also documentation generated with Doxygen.

Building

For Windows users, there is a Visual Studio 2019 solution file. Just open it and build any configuration you need (debug, release, 32-bit, 64-bit). You can also build everything using the BUILD.bat script.

For Unix users there is a makefile that creates a static library libutpp.a.
