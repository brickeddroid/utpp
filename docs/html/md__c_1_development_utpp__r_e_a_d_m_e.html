Building

For Windows users, there is a Visual Studio 2017 solution file. Just open it and build any configuration you need (debug, release, 32-bit, 64-bit).

For Unix users there is a makefile that creates a static library libutpp.a. For 32-bit version add ARCH=x86 to make command line. For debug version add _DEBUG=1.
